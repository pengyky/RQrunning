# Sylas Training Hub

🏃 一站式跑步训练强度计算器 - 融合心率区间与配速课表设计

## 功能特性

- **心率区间计算** - 基于储备心率法，可视化展示训练强度、呼吸商RQ和供能系统
- **配速课表设计** - 基于10km PB配速，生成间歇训练和比赛成绩预测
- **智能联动** - 心率与配速对照，一站式训练方案
